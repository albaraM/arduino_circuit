# arduino_circuit

## Tools :
1- Arduino Uno

2- 5 servo motor

3- 5 volt battery

4- Wires

5- breadboard


## Steps:
1- We connected the motor to the Arduino

2- We used ports 11, 10, 9, 6 and 3

3 - We connected the power to the motor with 5 volts

4- We connected  the motor on the ground

5- We powered the Arduino
